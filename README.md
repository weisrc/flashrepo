# Flashrepo

A simple game Flash game repository manager and player built with Tauri and the Ruffle Flash emulator.

## Admin Mode

Set `FLASHREPO_ADMIN` environment variable to `true` to enable admin mode:
- Create tags
- Add games
- Edit games
- Delete games
- Delete tags

## Disclaimer

This is a weekend project. Expect bugs and missing features. Pull requests are welcome!
